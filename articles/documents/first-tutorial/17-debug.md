# [Ruby](#tab/lang-ruby)

`require "lumino"` の前に `$LUMINO_LOG_LEVEL` 変数を定義し、出力レベルを指定することでログを出力できるようになります。

```ruby
$LUMINO_LOG_LEVEL = 2
require "lumino"
```

| 値 | 意味 |
|---|---|
| 1 | VERBOSE |
| 2 | DEBUG |
| 3 | INFO |
| 4 | WARNING |
| 5 | ERROR |
| 6 | FATAL |
